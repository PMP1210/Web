Problem 38

By Programmer February 2, 2023

Problem statment:

Write a program to replace a character in given string.

Code:


                  
    #include <stdio.h>
    #include <string.h>
                        
    int main()
    {
        char str[100],ch,ch2;
        int i,signal=0;
                        
        printf("\n_______________________________________________\n\n");                
        printf("\tReplace_Character");
        printf("\n_______________________________________________\n\n");
                        
        printf("Enter Your String : ");
        gets(str);
                                        
        printf("\nEnter The Character To Replace : ");
        scanf(" %c",&ch);
                        
        printf("\nEnter The New Character To Replace %c : ",ch);
        scanf(" %c",&ch2);
                        
        printf("\n_______________________________________________\n\n");
                        
        for(i=0;str[i] != '\0';i++)
        {
            if(ch == str[i])
            {
                str[i] = ch2;
                signal = 1;
            }
        }
                        
        if(signal == 0)
        {
            printf("\n\n%c Is Not Found In Given String",ch);
        }
                        
        printf("\nNew String Is : ");
        puts(str);
                        
        printf("\n_______________________________________________\n\n");
                        
        return 0;
    }

Description:

This code is written in C and aims to replace a specific character with a new character in a given string. Let's go through the code step by step:

1. The code starts with including the standard input/output library `stdio.h` and the string library `string.h`, which is required for string-related functions.

2. The code defines the `main()` function, which is the entry point of the program.

3. The initial set of `printf` statements are used to print a header for the program.

4. The code declares a character array `str` of size 100 to store the user-entered string.

5. Character variables `ch` and `ch2` are declared to store the character to be replaced and the new character, respectively.

6. An integer variable `i` is declared to be used as a loop counter, and a signal variable `signal` is initialized to 0. This variable will be used to check if the character to be replaced is found in the string.

7. The code prompts the user to enter a string using `printf` and `gets` functions. Note that using `gets` is generally discouraged due to potential buffer overflow vulnerabilities. It's better to use `fgets` instead.

8. The code prompts the user to enter the character to be replaced using `printf` and `scanf` functions.

9. The code prompts the user to enter the new character using `printf` and `scanf` functions.

10. The code prints a separator.

11. A `for` loop is used to iterate through each character in the string until the null terminator (`'\0'`) is encountered.

12. Inside the loop, an `if` statement checks if the current character `ch` matches the character in the string `str[i]`.

13. If a match is found, the code assigns the new character `ch2` to `str[i]` to replace the old character and sets the `signal` variable to 1 to indicate that the character was found and replaced.

14. After the loop, an `if` statement checks the value of `signal`. If it is still 0, it means the character to be replaced was not found in the string. In that case, the code prints a message indicating that the character was not found.

15. The code prints the modified string using the `puts` function.

16. The code prints a separator.

17. The `return 0;` statement is used to exit the `main()` function and terminate the program.

It then prints the modified string. If the character to be replaced is not found, it displays a message indicating that it was not found.
