Problem 02

By Programmer February 2, 2023

Problem statment:

Write a program to find area of triangle (a=h*b*.5) a = area h = height b = base

Code:


                  
#include <stdio.h>

int main()
{
  float a, h, b;
  
  printf("\n_____________________________________");
  printf("\tArea Of Triangle ");
  printf("\n_____________________________________");
                      
  printf("\n\nEnter The Height : ");
  scanf("%f", &h);
                      
  printf("\nEnter The Base : ");
  scanf("%f", &b);
                      
  a = h * b * .5;
                      
  printf("\n_____________________________________");
  printf("\n\nArea Of Triangle : %.2f Sq. Units", a);
  printf("\n\nFormula Used (a=h*b*.5)");
  printf("\n_____________________________________\n\n");
                      
  return 0;
 }

Description:

The provided code is a C program that calculates the area of a triangle based on user input for the height and base of the triangle. Here is a description of the code:

1. The necessary header file, `stdio.h`, is included to enable input and output operations.

2. The `main()` function is defined as the entry point of the program.

3. Three variables of type `float` are declared: `a` for storing the area of the triangle, `h` for storing the height, and `b` for storing the base.

4. Several `printf()` statements are used to display a heading and prompts to the user for input.

5. The user is asked to enter the height of the triangle using the `scanf()` function, and the value is stored in the variable `h`.

6. The user is then asked to enter the base of the triangle using `scanf()`, and the value is stored in the variable `b`.

7. The area of the triangle is calculated by multiplying the height (`h`), base (`b`), and 0.5 (to calculate half of the product) and assigning the result to the variable `a`.

8. Additional `printf()` statements are used to display the calculated area of the triangle and the formula used for the calculation.

9. The program ends with `return 0`, indicating successful execution.

In summary, this code prompts the user for the height and base of a triangle, calculates its area using the formula `area = 0.5 * height * base`, and displays the result along with the formula used.
