By Programmer February 2, 2023

Problem statment:

Write a C program to find 1+1/2+1/3+1/4+....+1/n

Code:


                  
    #include <stdio.h>

    int main()
    {
        int n,i;
        float sum=0;
                        
        printf("\n_______________________________________________\n\n");
        printf("\n\tEvalute The Series 1/1 +1/2 +1/3 +1/4+....+1/n");
        printf("\n_______________________________________________\n\n");
        printf("\n\nEnter The Value Of N : ");
        scanf("%d",&n);
                        
        for(i=1; i<=n; i++)
        {                
            sum = sum + (1.0/i);
        }
                        
        printf("\n\n_______________________________________________________________");
        printf("\n\nSum Of %d Terms Is : %.2f",n,sum);
        printf("\n_______________________________________________________________\n\n");
                        
        return 0;
    }

Description:

1. The code starts with including the standard input-output library `stdio.h`.

3. The program displays a title and description using `printf()` to indicate the purpose of the program.

4. Integer variable `n` and float variable `sum` are declared. `n` represents the number of terms in the series, and `sum` stores the sum of the series.

5. The user is prompted to enter the value of `n` using the `printf()` statement. The entered value is stored in the variable `n` using `scanf()`.

6. A `for` loop is used to iterate from `i = 1` to `i < = n` to calculate the sum of the series.

7. Inside the loop, the expression `(1.0/i)` calculates the reciprocal of `i`, and the result is added to the `sum` variable using the assignment operator `+=`.

8. After the loop completes, the program displays the sum of the series using `printf()`. The format specifier `%.2f` is used to display the sum with two decimal places.

9. The `main()` function returns 0, indicating successful execution of the program.

The code allows the user to enter the value of `n` and calculates the sum of the series 1/1 + 1/2 + 1/3 + 1/4 + ... + 1/n using a `for` loop. The result is then displayed with two decimal places using `printf()`.
