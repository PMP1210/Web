By Programmer February 2, 2023

Problem statment:

Write a program to convert string into upper case

Code:


                  
    #include <stdio.h>
    #include <string.h>
                        
    int main()
    {
        printf("\n_______________________________________________\n\n");                
        printf("\tConvert To Uppercase");
        printf("\n_______________________________________________\n\n");
                        
        char str[50];
                        
        printf("Enter String : ");
        gets(str);
                        
        printf("\nYour String Is : ");
        puts(str);
                        
        printf("\n_______________________________________________\n\n");
        printf("\nUppercase String Is : %s",strupr(str));
        printf("\n_______________________________________________\n\n");
                        
        return 0;
    }

Description:

Let's go through the code step by step and understand its functionality:

1. The code begins with the inclusion of necessary header files: `stdio.h` for input/output operations and `string.h` for string manipulation functions.

2. The `main()` function is defined, which is the entry point of the program.

3. The series of `printf()` statements are used to display a header and instructions to the user.

4. An array of characters called `str` is declared, with a size of 50. This array will be used to store the user's input string.

5. The `printf()` function prompts the user to enter a string.

6. The `gets()` function is used to read the string entered by the user and store it in the `str` array. Note that using `gets()` is generally discouraged due to security risks associated with buffer overflows. A safer alternative would be to use `fgets()`.

7. The `printf()` function is used to display the original string entered by the user using the `puts()` function.

8. The code then uses the `printf()` function to display a separator line.

9. The `strupr()` function is called to convert the characters in the `str` array to uppercase. This function modifies the original string in place.

10. The uppercase string is then displayed using the `printf()` function.

11. The code displays another separator line.

12. Finally, the `return 0` statement signifies the end of the `main()` function and the program's execution.

Overall, this code takes a string as input from the user, converts it to uppercase using the `strupr()` function, and displays the uppercase string.
