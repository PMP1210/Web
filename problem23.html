By Programmer February 2, 2023

Problem statment:

Read five persons height and weight and count the number of person having height greater than 170 and weight less than 50,

Code:


                  
    #include <stdio.h>

    int main()
    {                    
        printf("\n_______________________________________________\n\n");
        printf("\tNumber Of Person Having Height Greater Than 170 And Weight Less Than 50");
        printf("\n_______________________________________________\n\n");                       
        
        int height,weight,i,n=0;
                        
        for (i=0;i< 5;i++)
        {
            printf("\n\nEnter The Height Of Person [%d] : ",i+1);
            scanf("%d",&height);
                        
            printf("\nEnter The Weight Of Person [%d] : ",i+1);
            scanf("%d",&weight);
                        
            printf("____________________________________________");
                        
            if(height>170 && weight< 50)
            {
                n++;
            }
        }
                        
        printf("\n\n______________________________________________________________________________");
        printf("\n\n%d Person Having Height Greater Than 170 And Weight Less Than 50",n);
        printf("\n\n______________________________________________________________________________\n\n");
        return 0;
    }

Description:

The given code is a C program to count the number of people who have a height greater than 170 and a weight less than 50. Here's a breakdown of the code:

1. The code starts with including the standard input-output library `stdio.h`.

2. The `main()` function is defined, which serves as the entry point for the program.

3. The program displays a title and description using `printf()` to indicate the purpose of the program.

4. Integer variables `height`, `weight`, `i`, and `n` are declared. `height` stores the height of each person, `weight` stores the weight of each person, `i` is a loop counter, and `n` stores the count of people who satisfy the condition.

5. A `for` loop is used to iterate through each person. The loop iterates 5 times to represent 5 people.

6. Inside the loop, the user is prompted to enter the height of a person using the `printf()` statement. The entered height is stored in the variable `height`.

7. The user is then prompted to enter the weight of the person using the `printf()` statement. The entered weight is stored in the variable `weight`.

8. The line `if(height>170 && weight< 50)` checks if the height is greater than 170 and the weight is less than 50. If the condition is true, the variable `n` is incremented by 1.

9. After the loop completes for all 5 people, the program displays a separator line and the count of people who satisfy the condition using `printf()`.

10. The `main()` function returns 0, indicating successful execution of the program.

The code allows the user to enter the height and weight of 5 people. It counts the number of people who have a height greater than 170 and a weight less than 50. The count is displayed at the end of the program using `printf()`.
