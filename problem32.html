Problem 32

By Programmer February 2, 2023

Problem statment:

Write a C program to read and store the roll no and marks of 20 students using array.

Code:


                  
    #include <stdio.h>

    int main()
    {
        printf("\n_______________________________________________\n\n");                
        printf("\tMarks-Entry");
        printf("\n_______________________________________________\n\n");                       
        int students,subjects,i,j;
                        
        printf("\n\nEnter The Number Of Students : ");
        scanf("%d",&students);
                        
        printf("\nENter The Number Of Subjects : ");
        scanf("%d",&subjects);
                        
        int marks[students][subjects+1];
                        
        for(i=0;i< students;i++)
        {
            printf("\n\nEnter Roll No. : ");
            scanf("%d",&marks[i][0]);
                        
            for(j=0;j< subjects;j++)
            {
                printf("\nEnter Marks Of Subject [%d] : ",j+1);
                scanf("%d",&marks[i][j+1]);
            }
        }
                        
                        
        for(i=0;i< students;i++)
        {
                        
            printf("\n\nRoll No. : %d",marks[i][0]);
                        
            for(j=0;j< subjects;j++)
            {
                printf("\nMarks Of Subject [%d] : %d",j+1,marks[i][j+1]);
                        
            }
        }
                        
        return 0;
    }

Description:

This code is written in C and it allows the user to enter marks for multiple students and subjects and then displays the entered marks. Let's go through the code step by step:

1. The code starts with including the standard input/output library `stdio.h` which provides functions like `printf` and `scanf` for input/output operations.

2. The `main()` function is the entry point of the program and where the execution begins.

3. The first set of `printf` statements are used to print a header for the program.

4. The code declares four variables: `students`, `subjects`, `i`, and `j`. `students` and `subjects` are used to store the number of students and subjects, while `i` and `j` are loop control variables.

5. The code prompts the user to enter the number of students and subjects using `printf` statements and reads the values using `scanf`.

6. The code declares a 2D array called `marks` of size `students` by `subjects+1`, where each row represents a student and each column represents a subject. The extra column is used to store the roll number of each student.

7. The first `for` loop is used to iterate through each student.

8. Inside the first `for` loop, the code prompts the user to enter the roll number for the current student and stores it in the first column (`marks[i][0]`) of the `marks` array.

9. The second nested `for` loop is used to iterate through each subject for the current student.

10. Inside the nested `for` loop, the code prompts the user to enter the marks for the current subject and stores it in the corresponding cell (`marks[i][j+1]`) of the `marks` array.

11. After the user enters all the marks, the code enters another `for` loop to display the entered marks.

12. The loop iterates through each student.

13. Inside the loop, the code first prints the roll number of the current student using `printf`.

14. Then, the nested `for` loop is used to iterate through each subject for the current student.

15. Inside the nested `for` loop, the code prints the marks of each subject for the current student using `printf`.

16. Finally, the `return 0;` statement is used to exit the `main()` function and terminate the program.

In summary, this code allows the user to enter marks for multiple students and subjects, stores the entered marks in a 2D array, and then displays the entered marks along with the corresponding roll numbers.
