Problem 29

By Programmer February 2, 2023

Problem statment:

Write a program to print diffrent star patterns.

Code:


                  
    #include <stdio.h>

    int main()
    {
        printf("\n_______________________________________________\n\n");
        printf("\tStar_Pattern");
        printf("\n_______________________________________________\n\n");
                        
        int i, j, k;
                        
        for (i = 1; i <= 5; i++)
        {
            for (j = 1; j <= i; j++)
            {
                printf("* ");
            }
            printf("\n");
        }
                        
        printf("\n_______________________________________________\n\n");
        for (i = 5; i >= 0; i--)
        {
            for (j = 1; j <= i; j++)
            {
                printf("* ");
            }
            printf("\n");
        }
                        
        printf("\n_______________________________________________\n\n");
                        
        for (i = 0; i < 5; i++)
        {
            for (k = 5; k > i; k--)
            {
                printf(" ");
            }
            for (j = 0; j < i; j++)
            {
                        
                printf("* ");
            }
            printf("\n");
        }
        
        return 0;
    }

Description:

3. The program displays a title and description using `printf()` to indicate the purpose of the program.

4. Three nested `for` loops are used to print different star patterns.

5. The first pattern is an upward-triangle pattern. The outer loop controls the number of rows, and the inner loop prints the stars in each row. The number of stars in each row is equal to the row number. After printing the stars, a new line is inserted using `printf("\n")`.

6. The second pattern is a downward-triangle pattern. It follows the same logic as the first pattern but starts with the maximum number of stars and decreases with each row.

7. The third pattern is a right-aligned pyramid pattern. It uses two nested loops. The outer loop controls the number of rows, and the inner loop prints spaces before printing the stars. The number of spaces decreases with each row, and the number of stars increases. After printing the stars, a new line is inserted.

8. The program displays a separator line using `printf()` to separate the patterns.

9. The `main()` function returns 0, indicating successful execution of the program.
